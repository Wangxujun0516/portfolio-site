API Quick Start

This sample shows how I structure a concise onboarding guide for developers who want to make their first successful API request quickly.

Goal

Use the Fictional Text API to generate a short text response in less than five minutes.

Before You Begin

  • Create an account in the Fictional Text console.
  • Generate an API key from Settings > API Keys.
  • Install curl or use your preferred HTTP client.

Make Your First Request

Replace YOUR_API_KEY with your actual key.

curl -X POST https://api.example.dev/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "writer-lite",
    "prompt": "Write a welcome message for a new user.",
    "max_tokens": 80
  }'

Example Response

{
  "id": "resp_12345",
  "model": "writer-lite",
  "output": "Welcome! We are glad you're here. Let's get you started.",
  "usage": {
    "input_tokens": 9,
    "output_tokens": 13
  }
}

What to Do Next

  • Review the authentication guide.
  • Try changing the model or prompt.
  • Move on to the SDK integration guide for application-level usage.
